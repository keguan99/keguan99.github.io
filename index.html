对于这次夏令营的感受

  • “害怕人言，就不敢勇往直前；害怕嘲讽，就不敢坚守信念。我相信：一个人心在哪里，哪里就有风景；志在哪里，哪里就有成功：梦在哪里，哪里就有未来。”
  • 我有幸参加了这次学校组织的IT训练营，抱着期待和学习的心情，我参与其中，受益良多，给我的人生中上了生动的一课。
  • 我们这次训练营是学校邀请宏福集团的老师来讲课的，在这次训练营中认识了许多人，每个人都是那么的和蔼可亲。大家都全身心的投入其中，互帮互助，展示出了最精彩的自己，因此这次训练营拉近了彼此的距离。老师也不是严肃敬畏的，而是幽默风趣的，同学也不是生疏冷漠的，而是热情热心的。
  • 在这次训练营中我学到了很多知识，对于我以后在IT方面有很大的帮助。
  • 从现在开始我将把这次训练营中所学到的知识好好运用到以后的学习中。
  • 总结：1把这次训练营所学到到知识和同学们互相分享；2未来的日子中我会以在训练营积极学习的方式运用到平常学习中，不断总结完善自己；3努力学习，天天向上，乐观面对一切困难。

Launch of blog

hexo博客的上线

  • 1 进入https://github.com网站，在右侧提示写上用户名～邮箱～密码点击Sign up for GitHub，接下来按提示操作即可。
  • 2 进入之前写的邮箱，会收到GitHub发来的邮件，打开之后点击Read the guide。
  • 3 点击Read the guide之后会进入到Create a new repository页面，只需在此页面填上Repository name即可。
  • 4 在CentOS 7中找到config。yml文件夹打开在最后一行type后添加git，下一行添加repository： http：//github.com/keguan99/keguan99.github.io.git，在下一行添加branch： master
  • 5 cd blog
  • 6 sudo npm install hexo-deployer-git –save
  • 7 git config –global user.email”你的邮箱”
  • 8 git config –global user.name”你的用户名”
  • 9 gedit_config.yml
  • 10 hexo g
  • 11 hexo d

    hexo博客上线后博客中文章的更新

  • 1 cd blog
  • 2 hexo new “title”
  • 3 gedit ~/…..md
  • 4 hexo g
  • 5 hexo d

Operations in Virtual machines

虚拟机的删除

  • 1点页面中的虚拟机选择管理，点击“从磁盘删除”即可，牢记这种删除方式是不可逆的。
  • 2点页面中的虚拟机选择快照，也可以删除，是可以回复的

    虚拟机里文件的创建与删除

    一

  • 1mkdir | make directory |创建目录
  • mkdir DIRNAME
  • 2ls | list |显示目录中的文件
  • ls
  • 3touch | touch |创建文件
  • touch FILENAME
  • 4rm | remove |删除文件
  • rm FILENAME
  • 5cd | change directory |切换目录
  • cd DIRNAME

    二

  • yum list | 显示软件列表
  • yum install APPNAME | 安装指示软件
  • yum search APPNAME | 搜索相关软件
  • yum check-update | 检查是否需要更新
  • yum update APPNAME | 更新指定软件
  • yum update | 更新所有软件
  • yum remove APPNAME | 卸载指定软件
