Virtual machine installation

##今天一共讲了虚拟机的安装和4个不同虚拟机系统的安装。

##分别是windows 10,windows 7,centOS 7和deepin四个虚拟机系统。

#虚拟机的安装
找到并打开文件

仔细阅读最终用户许可协议，并点击（我接受许可协议中的条款），点击下一步

选择安装位置并点击下一步

编辑默认设置并点击下一步

选择要放入系统的快捷方式并点击下一步

出现下图就说明已准备好安装VMware Workstarion Pro，点击安装

安装中，请等待

安装向导已完成，点击许可证

输入许可证秘钥


出现下图说明操作成功，点击完成即可
