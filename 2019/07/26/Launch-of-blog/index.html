Launch of blog

hexo博客的上线

  • 1 进入https://github.com网站，在右侧提示写上用户名～邮箱～密码点击Sign up for GitHub，接下来按提示操作即可。
  • 2 进入之前写的邮箱，会收到GitHub发来的邮件，打开之后点击Read the guide。
  • 3 点击Read the guide之后会进入到Create a new repository页面，只需在此页面填上Repository name即可。
  • 4 在CentOS 7中找到config。yml文件夹打开在最后一行type后添加git，下一行添加repository： http：//github.com/keguan99/keguan99.github.io.git，在下一行添加branch： master
  • 5 cd blog
  • 6 sudo npm install hexo-deployer-git –save
  • 7 git config –global user.email”你的邮箱”
  • 8 git config –global user.name”你的用户名”
  • 9 gedit_config.yml
  • 10 hexo g
  • 11 hexo d

    hexo博客上线后博客中文章的更新

  • 1 cd blog
  • 2 hexo new “title”
  • 3 gedit ~/…..md
  • 4 hexo g
  • 5 hexo d
